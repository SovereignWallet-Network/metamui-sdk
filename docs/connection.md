## buildConnection(network, ignoreCache)
Return an APIPromise object

**Kind**: global function  

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| network | <code>string</code> | <code>&quot;local&quot;</code> | MetaMUI network provider to connect |
| ignoreCache | <code>boolean</code> | <code>false</code> | (optional) (default=true) Note : setting the ignoreCache value to true will create a new ws ws conection on every call |
